## How to Use This Appendix

As you write chapters and cite scripture:

1. **Add the reference** under the appropriate book
2. **Include a brief description** of how it's used
3. **Link to the chapter(s)** where it appears
4. **Group related references** together for easier navigation

### Format Example:
```markdown
- **Book Chapter:Verse** — Brief description
  - See: [Chapter X — Title](../part-XX/chapter-XX.md)
  - See: [Chapter Y — Title](../part-YY/chapter-YY.md) *(if discussed in multiple chapters)*
```
